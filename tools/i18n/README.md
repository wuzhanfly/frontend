# 国际化工具集

这是一套用于项目国际化的工具脚本，包含以下功能：

1. **键值对提取** - 从源代码中提取需要国际化的文本
2. **语言包生成** - 根据提取的键值对生成多语言翻译文件
3. **国际化组件导入** - 自动为组件添加国际化依赖
4. **文本替换** - 将硬编码文本替换为国际化键

## 使用流程

1. 运行 `extract-keys.js` 提取项目中的键值对
2. 运行 `generate-locales.js` 生成语言包文件
3. 运行 `setup-i18n-components.js` 为组件添加国际化支持
4. 运行 `replace-text-with-keys.js` 替换硬编码文本

## 注意事项

- 所有脚本都会在执行前备份原始文件
- 可以通过配置文件调整提取和替换规则
- 脚本支持增量更新，不会覆盖已有的翻译
